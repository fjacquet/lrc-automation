<task type="auto">
  <name>Task 2: Rewrite ci.yml — 3-OS matrix, uv run commands, setup-uv@v7</name>
  <files>.github/workflows/ci.yml</files>
  <action>
Replace the entire contents of `.github/workflows/ci.yml` with the following:

```yaml
name: CI

on:
  push:
    branches: [main]
  pull_request:
    branches: [main]

jobs:
  check:
    runs-on: ${{ matrix.os }}
    strategy:
      fail-fast: false
      matrix:
        os: [ubuntu-latest, macos-latest, windows-latest]
        python-version: ["3.12", "3.13"]
    steps:
      - uses: actions/checkout@v4

      - uses: astral-sh/setup-uv@v7
        with:
          python-version: ${{ matrix.python-version }}
          enable-cache: true

      - run: uv sync

      - run: uv run ruff check .
      - run: uv run ruff format --check .
      - run: uv run mypy src/
      - run: uv run pytest -v
```

Key changes from original:

- `runs-on: ubuntu-latest` replaced by `runs-on: ${{ matrix.os }}` with os matrix dimension
- Added `os: [ubuntu-latest, macos-latest, windows-latest]` to matrix
- `fail-fast: false` so a failure on one platform does not cancel others
- `setup-uv@v4` bumped to `setup-uv@v7` with `enable-cache: true` (OS-aware cache keys)
- `make check` replaced by four individual `uv run` commands that work on all three platforms
- Do NOT use `uv sync --all-extras` — `reverse_geocoder` has no Windows wheel and would break the Windows runner
  </action>
